### BOGGLE
+ 실천형 수질 관리 애플리케이션 (➡️Name 'BOGGLE' is the onomatopoeia that explan bubble's shape in Korean.)
+ 2024.03.02 ~ 2024.06.20
+ 김민, 구경선, 예경민, 윤별
  
### 사용 기술 및 환경
+ PYTHON
+ Django
+ Dart
+ Fultter
+ Android Studio
+ VScode

### What is this app?

This is the app that make you move to save water from the water pollution!

you can certify your detergent and sewer to get points!
<br/>

### 공모전 참여
공모전 참여로 인한 기능 변경(물 빅테이터 공모전, 공개SW 개발자대회)
  * 기존 기능
    - OCR 기능을 사용하여 세제의 text 추출
    - 객관식 퀴즈
<br/>

  * 추가, 변경된 기능
    - YOLO를 학습하여 친환경 마크 탐지로 세제인증 진행
    - 주관식 퀴즈 삽입
    - 물과 관련된 여행지 추천
    - 빗물받이 GPS 인식

<br/>
